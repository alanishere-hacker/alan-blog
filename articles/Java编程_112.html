Java集合框架与泛型：数据去重与排序实践

一、实验目的及要求
1、掌握根据不同的应用情况选择不同的容器。
2、掌握容器的常用方法来完成操作。
3、学习泛型的定义和使用方法。
二、实验原理与内容
1、给定一个字符串数组，去掉重复的字符串，然后将去重后的所有字符串排序输出。请给出至少2种方法（其中至少一种需要用到容器）。
示例：给定数组String [] strings = {"Monday","Sunday","Tuesday", "Wednesday","Thursday","Friday","Saturday","Wednesday","Sunday"};
输出“Friday, Monday, Sunday, Saturday, Thursday, Tuesday, Wednesday”。
实验二是矩阵元素是整数的加法和乘法，使用泛型，使所编写的程序可实现矩阵元素是整数和浮点数的加法和乘法，并输出运算结果。

1.代码思路：
实验一方法一（使用 Set 去重并排序）：
输入处理：
程序首先通过 Scanner 类从用户输入获取字符串的个数 n，并通过 scanner.nextLine() 读取换行符，确保后续读取字符串时不会受到换行符影响。
接着创建一个字符串数组 strings 来存储用户输入的字符串，并通过 for 循环读取并存储 n 个字符串。
去重并排序操作：
调用 removeDuplicatesAndSortUsingSet 方法，传入字符串数组 strings，目的是去重并排序。
去重操作：
在 removeDuplicatesAndSortUsingSet 方法中，首先将输入的字符串数组转为 LinkedHashSet。LinkedHashSet 是一种基于哈希表和链表的集合，它能够去除重复元素，并且保持元素的插入顺序。
通过 new LinkedHashSet<>(Arrays.asList(strings)) 将字符串数组转为集合，去除重复元素。
排序操作：
将去重后的 Set 转为 ArrayList，因为 List 可以使用 Collections.sort() 方法进行排序。
使用 Collections.sort(sortedList) 对去重后的字符串进行字典顺序排序。
输出：
排序完成后，利用 forEach 方法输出排序后的字符串列表，输出的顺序是字典顺序。
关闭资源：
最后，关闭 Scanner 对象以释放资源。
实验一方法二（使用 Stream 去重并排序）：
输入处理：
程序首先通过 Scanner 类从用户输入获取字符串的个数 n，并通过 scanner.nextLine() 处理换行符，确保后续读取字符串时不会受到换行符影响。
创建一个字符串数组 strings 用于存储用户输入的字符串，接着使用 for 循环来逐个读取用户输入的字符串并存储在数组中。
去重并排序操作：
程序调用 removeDuplicatesAndSortUsingStream 方法，传入用户输入的字符串数组 strings，目的是去重并排序。
使用 Stream 进行去重和排序：
在 removeDuplicatesAndSortUsingStream 方法中，首先通过 Arrays.stream(strings) 将字符串数组转换为 Stream。
使用 distinct() 方法去重，确保只保留每个字符串的唯一性。
使用 sorted() 方法对去重后的字符串进行字典顺序排序。
最后，使用 forEach(System.out::println) 输出排序后的结果，每个字符串逐行输出。
输出：
排序后的字符串通过 forEach 方法逐个打印到控制台，确保用户能够看到去重并排序后的结果。
关闭资源：
在程序结束时，使用 scanner.close() 来关闭 Scanner 对象，释放资源。
实验二：
输入处理：
首先，程序使用 Scanner 类获取用户输入的矩阵的行数 rows 和列数 cols。
然后，创建两个矩阵 intMatrix1 和 intMatrix2，这两个矩阵分别存储用户输入的整数矩阵元素。
用户依次输入每个矩阵的元素，通过双重循环来填充 intMatrix1 和 intMatrix2。
执行矩阵运算（整数矩阵）：
程序调用 performMatrixOperations 方法，传入整数类型的矩阵 intMatrix1 和 intMatrix2，并设置 forceFloatingPoint 为 false，表示后续输出结果为整数。
在 performMatrixOperations 方法中，首先计算两个矩阵的加法，将结果存储在 additionResult 矩阵中。
接着，计算两个矩阵的乘法。根据矩阵乘法的规则，计算每个位置的元素并存储在 multiplicationResult 矩阵中。
输入浮点数矩阵：
程序同样使用 Scanner 获取两个浮点数矩阵 doubleMatrix1 和 doubleMatrix2，用户输入的元素会存储在这两个矩阵中。
执行矩阵运算（浮点数矩阵）：
再次调用 performMatrixOperations 方法，传入浮点类型的矩阵 doubleMatrix1 和 doubleMatrix2，并设置 forceFloatingPoint 为 true，表示输出时强制显示为浮点数格式。
计算过程与整数矩阵的运算相同，进行矩阵加法和矩阵乘法。
格式化输出：
performMatrixOperations 方法中会调用 printFormattedMatrix 方法来输出矩阵的计算过程。
在 printFormattedMatrix 中，通过循环遍历每一行的矩阵，格式化输出加法或乘法运算过程，并在适当位置打印运算符（例如 + 或 *）。
通过 printMatrixRow 方法根据 forceFloatingPoint 标志来判断是否强制将结果格式化为小数输出。整数结果不带小数，浮点数结果保留一位小数。
关闭资源：
程序结束时，调用 scanner.close() 关闭 Scanner 对象，释放资源。
关键点：
矩阵加法： 两个矩阵的对应位置元素相加，结果存储在新的矩阵中。
矩阵乘法： 根据矩阵乘法规则，计算每个位置的元素，并将结果存储在新的矩阵中。
格式化输出： 在输出时，根据矩阵的数据类型（整数或浮点数）来格式化输出，并且使用指定格式输出每个计算步骤。

package Test05;
import java.util.*;
public class ExperimentOneMethodOne {
public static void main(String[] args) {
// 创建Scanner对象以接收用户输入
Scanner scanner = new Scanner(System.in);
// 提示用户输入字符串的个数
System.out.print("请输入字符串的个数: ");
int n = scanner.nextInt();
scanner.nextLine(); // 处理换行符
// 创建字符串数组来存储用户输入
String[] strings = new String[n];
// 提示用户输入字符串
System.out.println("请输入" + n + "个字符串：");
for (int i = 0; i < n; i++) {
strings[i] = scanner.nextLine();
}
// 调用方法使用Set去重并排序
System.out.println("去重并排序的输出（使用Set）：");
removeDuplicatesAndSortUsingSet(strings);
// 关闭Scanner
scanner.close();
}
// 使用Set去重并排序
private static void removeDuplicatesAndSortUsingSet(String[] strings) {
// 将数组转为LinkedHashSet以去重并保持插入顺序
Set<String> uniqueStrings = new LinkedHashSet<>(Arrays.asList(strings));
// 将Set转换为ArrayList以进行排序
List<String> sortedList = new ArrayList<>(uniqueStrings);
Collections.sort(sortedList); // 使用Collections.sort()进行排序
// 输出排序后的列表
sortedList.forEach(System.out::println);
}
}

private static void removeDuplicatesAndSortUsingStream(String[] strings) {

.sorted() // 使用sorted()方法排序

package Test05;
import java.util.Scanner;
public class ExperimentTwo {
public static void main(String[] args) {
// 创建Scanner对象以接收用户输入
Scanner scanner = new Scanner(System.in);
// 输入矩阵的行数和列数
System.out.print("请输入矩阵的行数: ");
int rows = scanner.nextInt();
System.out.print("请输入矩阵的列数: ");
int cols = scanner.nextInt();
scanner.nextLine(); // 处理换行符
// 创建矩阵1和矩阵2
System.out.println("请输入第一个矩阵的元素：");
Integer[][] intMatrix1 = new Integer[rows][cols];
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
System.out.printf("请输入矩阵1第%d行第%d列的元素: ", i + 1, j + 1);
intMatrix1[i][j] = scanner.nextInt();
}
}
System.out.println("请输入第二个矩阵的元素：");
Integer[][] intMatrix2 = new Integer[rows][cols];
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
System.out.printf("请输入矩阵2第%d行第%d列的元素: ", i + 1, j + 1);
intMatrix2[i][j] = scanner.nextInt();
}
}
// 执行矩阵运算（整数矩阵）
System.out.println("测试矩阵元素是整数的输出：");
performMatrixOperations(intMatrix1, intMatrix2, false);
// 输入浮点数矩阵
System.out.println("\n请输入浮点数矩阵1的元素：");
Double[][] doubleMatrix1 = new Double[rows][cols];
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
System.out.printf("请输入浮点数矩阵1第%d行第%d列的元素: ", i + 1, j + 1);
doubleMatrix1[i][j] = scanner.nextDouble();
}
}
System.out.println("请输入浮点数矩阵2的元素：");
Double[][] doubleMatrix2 = new Double[rows][cols];
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
System.out.printf("请输入浮点数矩阵2第%d行第%d列的元素: ", i + 1, j + 1);
doubleMatrix2[i][j] = scanner.nextDouble();
}
}
// 执行浮点数矩阵运算
System.out.println("\n测试矩阵元素是浮点数的输出：");
performMatrixOperations(doubleMatrix1, doubleMatrix2, true);
// 关闭Scanner
scanner.close();
}
/**
* 计算矩阵的加法、乘法并格式化输出
* @param matrix1 矩阵1
* @param matrix2 矩阵2
* @param forceFloatingPoint 是否强制显示为小数
*/
private static <T extends Number> void performMatrixOperations(T[][] matrix1, T[][] matrix2, boolean forceFloatingPoint) {
int rows = matrix1.length;
int cols = matrix1[0].length;
// 矩阵加法和乘法结果
Number[][] additionResult = new Number[rows][cols];
Number[][] multiplicationResult = new Number[rows][cols];
// 计算矩阵加法
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
double value1 = matrix1[i][j].doubleValue();
double value2 = matrix2[i][j].doubleValue();
additionResult[i][j] = value1 + value2; // 加法
}
}
// 计算矩阵乘法（内层修改，遵循矩阵乘法规则）
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
double sum = 0;
for (int k = 0; k < cols; k++) {
sum += matrix1[i][k].doubleValue() * matrix2[k][j].doubleValue(); // 矩阵乘法公式
}
multiplicationResult[i][j] = sum; // 存储乘法结果
}
}
// 格式化输出：矩阵加法
printFormattedMatrix(matrix1, matrix2, additionResult, "+", forceFloatingPoint);
// 格式化输出：矩阵乘法
printFormattedMatrix(matrix1, matrix2, multiplicationResult, "*", forceFloatingPoint);
}
/**
* 格式化输出矩阵的计算过程
* @param matrix1 矩阵1
* @param matrix2 矩阵2
* @param result 计算结果矩阵
* @param operator 运算符（+ 或 *）
* @param forceFloatingPoint 是否强制显示为小数
*/
private static void printFormattedMatrix(Number[][] matrix1, Number[][] matrix2, Number[][] result, String operator, boolean forceFloatingPoint) {
int rows = matrix1.length;
for (int i = 0; i < rows; i++) {
// 打印矩阵1的一行
printMatrixRow(matrix1[i], forceFloatingPoint);
if (i == rows / 2) {
System.out.print(" " + operator + " ");
} else {
System.out.print(" ");
}
// 打印矩阵2的一行
printMatrixRow(matrix2[i], forceFloatingPoint);
if (i == rows / 2) {
System.out.print(" = ");
} else {
System.out.print(" ");
}
// 打印结果矩阵的一行
printMatrixRow(result[i], forceFloatingPoint);
System.out.println();
}
System.out.println();
}
/**
* 打印矩阵的一行，根据是否强制小数格式化输出
* @param row 矩阵的一行
* @param forceFloatingPoint 是否强制显示为小数
*/
private static void printMatrixRow(Number[] row, boolean forceFloatingPoint) {
for (Number num : row) {
if (forceFloatingPoint) {
System.out.printf("%6.1f", num.doubleValue()); // 强制显示为浮点数，保留1位小数
} else if (num instanceof Integer || num.doubleValue() % 1 == 0) {
System.out.printf("%4d", num.intValue()); // 输出整数
} else {
System.out.printf("%6.1f", num.doubleValue()); // 输出浮点数
}
}
}

二、实验原理与内容
1、从键盘输入一个整数，设计一个程序判断该整数是否是完全数。完全数是指其所有因素（包括1但不包括其自身）的和等于该数自身的数。例如28=1+2+4+7+14就是一个完全数。
2、水仙花数是指个位、十位和百位三个数的立方和等于这个三位数本身的数（提示：水仙花数涉及个位、十位、百位数字，所以水仙花数是一个三位数）。例如153=13+53+33。水仙花数只是的一种。如果在一个固定的中，一个n位等于自身各个数位上数字的n次幂之和，则称此数为自幂数。
附：十进制中十位以内自然数的自幂数名字：
一位自幂数：独身数
两位自幂数：不存在
三位自幂数：水仙花数
四位自幂数：四叶玫瑰数，如1634=14+64+34+44
五位自幂数：五角星数
六位自幂数：六合数
七位自幂数：北斗七星数
八位自幂数：八仙数
九位自幂数：九九重阳数，如146511208=19+49+69+59+19+19+29+09+89
十位自幂数：十全十美数
请编程实现输出以上所有的自幂数。
设计一个递归方法int binarySearch(int a [],int first, int last, int target)，用于查找给定的数值target是否在已排序的数组a中，其中first是数组a的第一个元素，last是数组a的最后一个元素，方法返回target在a中的位置，如果target不在a中，则返回-（insert_point+1），其中insert_point是要将target插入到a中的位置。示例输出：
数组a：[1, 3, 4, 8, 13, 20, 26, 88, 120]
20在a中的位置：5
21在a中的位置：-7
121在a中的位置：-10
上例中，20在数组a的第5个位置。21不在数组中，如果要将21插入数组a中，-7=-(insert_point+1），插入的位置为6。
提示：设计程序时要考虑数组a是升序还是降序。

}

System.out.println(num1+"为完全数");

package test01;
import java.util.*;
import java.io.*;
import java.util.Arrays;
public class Test1_3 {
public static void main(String[] args) throws IOException {
int[] arr;
int num1, m, n;
int first, last, target, length;
Scanner reader = new Scanner(System.in); // 创建一个Scanner对象用于读取用户输入
System.out.print("请输入数组的长度为：");
length = reader.nextInt(); // 读取用户输入的数组长度并赋值给length
arr = new int[length]; // 根据length创建一个新的整型数组arr
System.out.print("请输入数组元素：");
for (int i = 0; i < length; i++) { // 遍历数组，接收用户输入的每个元素
arr[i] = reader.nextInt(); // 将用户输入的元素赋值给数组arr的第i个位置
}
Arrays.sort(arr); // 对数组arr进行排序
System.out.print("该数组元素为：");
for (int a : arr) // 使用增强for循环遍历数组arr
System.out.println(a + " "); // 输出数组元素并在每个元素后添加空格
first = 0; // 初始化first为0，表示数组的第一个元素的索引
last = (length - 1); // 初始化last为length-1，表示数组的最后一个元素的索引
System.out.println("数组的第一个元素为：" + arr[first]);
System.out.println("数组的最后一个元素为：" + arr[last]);
System.out.print("请输入要查找的元素：");
target = reader.nextInt(); // 读取用户输入的目标值并赋值给target
System.out.println(target + "在x的位置为：" + binarySearch(arr, first, last, target)); // 调用binarySearch()方法查找目标值，并输出结果
}
// 定义一个递归函数binarySearch()，用于在已排序的数组arr中查找目标值target
public static int binarySearch(int[] x, int first, int last, int target) {
if (first > last) { // 如果first大于last，说明目标值不存在于数组中
return (-first - 1); // 返回一个负数表示未找到目标值
}
int mid = (int) (first + last) / 2; // 计算中间索引mid
int arr1 = x[mid]; // 获取数组arr中索引为mid的元素
if (target > x[mid]) { // 如果目标值大于中间元素，则在右半部分继续查找
return binarySearch(x, mid + 1, last, target);
} else if (target < x[mid]) { // 如果目标值小于中间元素，则在左半部分继续查找
return binarySearch(x, first, mid - 1, target);
} else return mid; // 如果目标值等于中间元素，则返回中间索引mid
}
}

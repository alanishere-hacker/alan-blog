二、实验原理与内容 1. 算法原理
本次实验采用RGB 图像最低有效位（LSB）隐写算法：
像素 R/G/B 分量用 8 位表示，最低位（第 0 位）对视觉影响最小。
将秘密信息按位替换像素最低位，人眼无法察觉差异。
提取时读取所有像素最低位，按编码还原文本信息。

from PIL import Image
import numpy as np
def encode_message(carrier_img, msg):
    img = carrier_img.convert('RGB')
    arr = np.array(img)
    flat = arr.flatten()
    msg_bytes = msg.encode('utf-8') + b'\xff'
    binary = ''.join(format(byte, '08b') for byte in msg_bytes)
    if len(binary) > len(flat):
        raise ValueError("消息过长，无法嵌入")
    for i in range(len(binary)):
        flat[i] = (flat[i] & 0xFE) | int(binary[i])
    return Image.fromarray(flat.reshape(arr.shape))
def decode_message(stego_img):
    img = stego_img.convert('RGB')
    arr = np.array(img)
    flat = arr.flatten()
    binary = ''.join(str(p & 1) for p in flat)
    bytes_list = [binary[i:i+8] for i in range(0, len(binary), 8)]
    result = b''
    for b in bytes_list:
        byte = int(b, 2)
        if byte == 0xff:
            break
        result += bytes([byte])
    return result.decode('utf-8')
if __name__ == "__main__":
    # 生成载体
    carrier = Image.new('RGB', (200, 200), color='white')
    carrier.save("carrier.png")
    print("载体已生成：carrier.png")
    # 隐写
    secret = " "
    stego = encode_message(carrier, secret)
    stego.save("stego.png")
    print("隐写完成：stego.png")
    # 提取
    extracted = decode_message(stego)
    print("提取结果：", extracted)

2 调试过程与问题解决
编码长度溢出问题
问题：消息二进制长度超过像素数量，导致嵌入失败。
调试：限制消息长度为姓名 + 学号，使用 200×200 大尺寸载体。
解决：确保载体像素总数远大于消息比特数，程序正常运行。
提取信息乱码
问题：未设置结束标记，提取到多余数据导致乱码。
调试：添加0xff作为结束符，提取时遇到终止。
解决：提取结果准确无乱码。
图片模式错误
问题：使用 RGBA 模式导致通道异常。
调试：统一转为RGB 模式处理。
解决：嵌入与提取稳定。
